science.testing_utils.generate_dummy_spectra(central_spectra=500, 730, 380, width=100, bins=10000.0, optical_range=300, 1064)¶

Generates dummy spectra for debugging purposes. Outputs tuple of arrays, index 0 holds the wavelength and index 1 holds the intensity value. Useful when no spectrometers is present. :param central_spectra: integer list of wavelengths to center on :param width: width of individual line :param bins: number of samples :param optical_range: range of values to be generated :return: tuple of arrays. index 0 holds wavelengths, index 1 holds intensity data
